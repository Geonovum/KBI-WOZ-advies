# Implementatie-advies modernisering gegevensaanlevering LV-WOZ

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC_BY_4.0-lightgrey.svg)](https://creativecommons.org/licenses/by/4.0/)

Deze repository bevat een adviesdocument over de modernisering van de gegevensaanlevering door
gemeenten aan de Landelijke Voorziening WOZ (LV-WOZ). Het document is opgesteld binnen het project
[Kennisborging en implementatieondersteuning](https://developer.overheid.nl/implementatie-ondersteuning)
van [Geonovum](https://www.geonovum.nl), in opdracht van het Ministerie van BZK.

Werkversie: https://geonovum.github.io/KBI-WOZ-advies/

## Over dit document

De huidige gegevensaanlevering aan de LV-WOZ is gebaseerd op StUF en ebMS2. Dit document analyseert
de transitie naar moderne alternatieven aan de hand van de WOZ-keten en formuleert
ontwerpuitgangspunten en aanbevelingen voor een REST-gebaseerde architectuur.

De analyse richt zich op het koppelvlak tussen gemeenten (als bronhouders) en de landelijke
voorziening. De bevindingen zijn breder toepasbaar op registraties die dezelfde transitie doormaken.
